# 🚀 건국대학교 수강신청 빈자리 알림 봇

건국대학교 수강신청 포털에 자동으로 로그인하여, 원하는 과목의 여석이 발생했을 때 텔레그램(Telegram)으로 실시간 알림을 보내주는 파이썬 자동화 스크립트입니다.

## ✨ 주요 기능
- **자동 로그인 (Auto Login)**: `requests.Session()`을 활용하여 재학생 통합 로그인(SID) 자동 수행
- **실시간 여석 감시**: 지정된 주기에 따라 백그라운드에서 여석(빈자리) 데이터 조회
- **텔레그램 알림**: 빈자리가 1개 이상 발생 시 즉시 스마트폰으로 푸시 알림 전송

## 🛠️ 사용 기술
- Python 3.x
- `requests`: HTTP 통신 및 세션(쿠키) 관리
- `python-dotenv`: 환경변수를 통한 개인정보 보호
